# วิธีการสร้าง Frame Template สำหรับ Photobooth

## 📐 ข้อกำหนด Frame Template

### ขนาดและรูปแบบ
- **ขนาดภาพ**: 1920 x 2560 pixels (แนวตั้ง) หรือขนาดที่ต้องการ
- **รูปแบบไฟล์**: PNG (ต้องมี alpha channel/โปร่งใส)
- **ชื่อไฟล์**: ตั้งชื่อตามใจชอบ เช่น `frame_classic.png`, `frame_birthday.png`

### การออกแบบ Frame

#### 1. สร้างใน Photoshop/GIMP/Figma
```
ขนาดแคนวาส: 1920 x 2560 px
Background: โปร่งใส (Transparent)

ตำแหน่งที่ต้องเว้นว่างสำหรับรูปภาพ:
- รูปที่ 1: X=160, Y=200,  ขนาด 1600x720 px
- รูปที่ 2: X=160, Y=970,  ขนาด 1600x720 px
- รูปที่ 3: X=160, Y=1740, ขนาด 1600x720 px
```

#### 2. Layer Structure (แนะนำ)
```
📁 Frame Template (1920x2560)
├── 🎨 Background Layer (สีพื้นหรือลวดลาย)
├── 📸 [เว้นว่าง] รูปที่ 1 (1600x720)
├── 🖼️  Border/Decoration รูปที่ 1
├── 📸 [เว้นว่าง] รูปที่ 2 (1600x720)
├── 🖼️  Border/Decoration รูปที่ 2
├── 📸 [เว้นว่าง] รูปที่ 3 (1600x720)
├── 🖼️  Border/Decoration รูปที่ 3
├── 🎭 Header (Title/Logo)
└── 📝 Footer (Text/Date)
```

#### 3. สร้าง Alpha Mask (สำหรับตำแหน่งรูป)
- **วิธีที่ 1**: วาดกรอบ Border แล้วเว้นส่วนกลางโปร่งใส
- **วิธีที่ 2**: ใช้ Layer Mask เพื่อกำหนดพื้นที่รูป
- **วิธีที่ 3**: ใช้โปรแกรมนี้จะวางรูปทับบน Frame อัตโนมัติ

## 💾 การใช้งาน

1. **บันทึก Frame**: วาง PNG ไฟล์ในโฟลเดอร์ `frames/`
2. **ตั้งค่าตำแหน่งรูป**: แก้ไขตำแหน่งใน `frame_config.json`
3. **เลือก Frame**: โปรแกรมจะอ่าน frame อัตโนมัติ

## 📝 ไฟล์ Config (frame_config.json)

สร้างไฟล์ `frames/frame_config.json`:

```json
{
  "default": {
    "frame_path": "frames/frame_classic.png",
    "image_positions": [
      {"x": 160, "y": 200, "width": 1600, "height": 720},
      {"x": 160, "y": 970, "width": 1600, "height": 720},
      {"x": 160, "y": 1740, "width": 1600, "height": 720}
    ],
    "canvas_size": [1920, 2560]
  },
  "birthday": {
    "frame_path": "frames/frame_birthday.png",
    "image_positions": [
      {"x": 200, "y": 250, "width": 1520, "height": 680},
      {"x": 200, "y": 1020, "width": 1520, "height": 680},
      {"x": 200, "y": 1790, "width": 1520, "height": 680}
    ],
    "canvas_size": [1920, 2560]
  }
}
```

## 🎯 Tips สำหรับการออกแบบ

1. **ใช้สีสดใส**: Frame ควรมีสีสันสดใสเพื่อความโดดเด่น
2. **เว้นพื้นที่รูป**: อย่าให้ decoration บังรูปหลัก
3. **ทดสอบ**: ลองวางรูปจริงเพื่อดูสัดส่วน
4. **เพิ่มข้อความ**: ใส่ชื่อ event, วันที่, hashtag
5. **Export PNG-24**: เพื่อรักษาคุณภาพและ transparency
